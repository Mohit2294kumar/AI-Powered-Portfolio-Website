# 🚀 AI-Powered Portfolio Website (MERN + Next.js)

A modern, fully responsive **AI-integrated portfolio website** built using **Next.js (React), Node.js APIs, MongoDB**, and deployed on Vercel.

This project showcases my skills, projects, and experience with an **interactive AI assistant**, smooth animations, and production-ready architecture.

---

## 🔥 Features

* ✨ Animated UI (Framer Motion + CSS)
* ⌨️ Typing Animation Hero Section
* 🔄 Auto-scrolling Project Marquee
* ⚡ Lazy Loading for performance optimization
* 🤖 AI Assistant (OpenAI + fallback support)
* 📬 Contact Form with MongoDB backend
* 🌐 Fully responsive design
* 🚀 Vercel deployment ready

## ⚙️ Installation & Setup

### 1. Clone Repository

```
git clone https://github.com/your-username/portfolio-ai-assistant.git
cd portfolio-ai-assistant
```

---

### 2. Install Dependencies

```
npm install
```

---

### 3. Setup Environment Variables

Create `.env.local` file:

```
MONGODB_URI=your_mongodb_connection_string
OPENAI_API_KEY=your_openai_api_key
```

> Note: AI works with fallback even without API key.

---

### 4. Run Locally

```
npm run dev
```

Open:

```
http://localhost:3000
```

---

## 🚀 Deployment (Vercel)

1. Push code to GitHub
2. Go to Vercel and import project
3. Add environment variables:

   * `MONGODB_URI`
   * `OPENAI_API_KEY`
4. Click **Deploy**

---

## ⚠️ Common Issues

### MongoDB Connection Error

* Allow IP access (`0.0.0.0/0`)
* Check credentials

### Build Error

```
npm run build
```

Fix errors before deploying.
